# Normal V2 Android APK

โปรเจกต์ Android WebView สำหรับเว็บไซต์:
https://texaspos99.github.io/Normal-V.2/

## สร้าง APK ด้วย GitHub Actions

1. อัปโหลดโฟลเดอร์ `android-apk` และ `.github` ไปที่ root ของ Repository
2. เปิดแท็บ Actions และเลือก `Build Normal V2 APK`
3. กด `Run workflow`
4. เมื่อเสร็จ ให้ดาวน์โหลด Artifact ชื่อ `Normal-V2-APK`

ไฟล์ติดตั้งอยู่ภายใน Artifact ชื่อ `app-debug.apk`
